Drone Motor Thrust Test Bench

Design and test of a motor thrust test bench || Data acquisition, visualization, and analysis of a MTTB

Project Details

In the realm of unmanned aerial vehicles (UAVs), the performance and efficiency of drone motors are critical factors that determine flight capabilities and overall operational success. To accurately assess these vital components, I developed a drone motor thrust test bench. This project involved carefully selecting materials based on availability and cost-effectiveness:

  • Arduino UNO R3: Served as the microcontroller, providing control and data processing capabilities.
  • Current sensor ACS712ELCTR-30A-T: Measured the current drawn by the motor to ensure accurate power consumption analysis.
  • Load cell TAL220: Used to measure the thrust generated by the motor, providing precise force data.
  • Amplifier module HX711: Amplified the signal from the load cell for accurate and reliable readings.
  • Motor Xing-E 2207 1800kv: The drone motor tested, chosen for its availability.
  • Readytosky 30A Brushless ESC 2-3S with Built-in 2A BEC: An electronic speed controller that regulated the motor's speed and power.
  • CNHL 30C 2200mAh 3S lipo battery: Provided the power for the motor and ESC.

The design process began by selecting key variables: thrust (grams of force) as the output and current (amperes) as the input. Using Autodesk Inventor Professional 2023, I created a detailed CAD model of the test bench, ensuring precise placement of all components based on thorough examination of propeller specifications. This meticulous design provides a robust foundation for accurately assessing drone motor performance.

The construction phase for the motor thrust test bench began with systematically acquiring all necessary components. Using provided data sheets, the team meticulously designed and precision-cut wood and MDF components, with special attention to crafting a motor base that ensured snug accommodation and robust integration into the overall assembly. The construction of the motor thrust test bench, highlights the successful execution of the design and construction processes. The wiring configuration, including the current sensor, amplifier module, and Electronic Speed Controller (ESC), was meticulously done according to manufacturer guidelines, with strategic pin allocation on the Arduino UNO ensuring precise and reliable data acquisition for thrust measurement analysis.

The project involved developing software in two interconnected phases. First, the Arduino UNO was programmed using the Arduino IDE, incorporating both intrinsic and the external libraries. This setup allowed the Arduino to process data from a current sensor and load cell while controlling motor power through an ESC, with precise timing. The data was formatted and transmitted to the serial port as a comma-separated list. The code executed a step test, modulating motor power within a 4000-millisecond timeframe, and balanced power management by reducing power levels more quickly than increasing them. The second phase involved managing the collected data, communicating between the serial port and Python to store data in a local PostgreSQL database. The data was then visualized using PowerBI, providing a comprehensive analysis of motor performance and efficiency.
