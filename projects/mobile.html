Mobile Robotics Navigation

Simulation, implementation, and autonomous mapping using ROS2

Presented at the 9th IEEE Colombian Conference on Applications of Computational Intelligence (ColCACI 2026) as part of Other Applications of Computational Intelligence Technical Session.

Project Details

Completed: June 2026

This project explores autonomous navigation from simulation to real-world implementation using ROS2 and Gazebo. The work investigates reactive navigation, map-based navigation, and autonomous mapping as complementary stages of a complete mobile robotics pipeline.

Simulation of Reactive and Map-Based Navigation

The first stage focused on developing a unified simulation framework capable of evaluating multiple autonomous navigation strategies under identical operating conditions. The environment was created in ROS2 Jazzy and Gazebo Harmonic using occupancy-grid maps and realistic robot models equipped with LiDAR and inertial sensing.

Reactive navigation algorithms including Wall Following, Distance-Based Follow-the-Gap, and TTC-Based Follow-the-Gap were evaluated alongside map-based approaches using ARA* global planning with Pure Pursuit and Stanley path tracking. All methods operated under a common Time-to-Collision Automatic Emergency Braking System, allowing direct comparison of navigation smoothness, safety, tracking performance, and traversal efficiency.

Implementation of Reactive and Map-Based Navigation

The simulation framework was subsequently transferred to a real Ackermann-steered RC platform equipped with a Raspberry Pi 5, RPLIDAR A1M8, IMU, wheel encoder, and distributed ROS2 architecture. The vehicle combines onboard computation with external processing for localization, visualization, and autonomous control while preserving real-time operation.

For map-based navigation, occupancy-grid maps generated with SLAM were used to compute global trajectories through the Anytime Repairing A* planner. These trajectories were tracked using an adaptive Pure Pursuit controller featuring velocity-dependent lookahead, heading-error-based speed adaptation, and encoder-informed odometry estimation. Experimental validation demonstrated reliable autonomous navigation across different operating speeds while maintaining stable trajectory convergence under real sensing and actuation constraints.

Reactive Mapping and Autonomous Exploration

The final stage addressed the fundamental challenge of autonomous map generation in previously unknown environments. Instead of relying on an existing map, the vehicle used a reactive Follow-the-Gap navigation strategy combined with SLAM Toolbox to safely explore and simultaneously construct occupancy-grid maps.

This approach demonstrates how reactive navigation naturally complements map-based planning by enabling the autonomous creation of the environmental representation required for global planning. During exploration, the robot relied exclusively on onboard LiDAR observations without predefined trajectories or external intervention, producing geometrically consistent maps that could later be reused for autonomous map-based navigation.

Project Contributions

Together, these projects establish a complete autonomous navigation workflow covering simulation, experimental implementation, and autonomous map generation. The work demonstrates how reactive navigation, map-based planning, and SLAM should be viewed as complementary technologies rather than competing approaches, providing a reproducible ROS2-based framework that spans the complete lifecycle of autonomous mobile robot deployment.
