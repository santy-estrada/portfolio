Geographic Data Visualization

Geographic data analysis and visualization using Plotly, Pandas, and Geopy for the Colombian delivery company Coordinadora

*Click the images from this page

Project Details

In my project with the delivery company Coordinadora, I focused on optimizing the efficiency and service quality of customer pickups and deliveries. A key aspect of this work involved addressing the complexities of managing geographical locations for these operations. Precise knowledge of package pickup and delivery locations is crucial, but inconsistencies in address nomenclature and structure across various cities, combined with limited coverage in digital map services, make this task challenging.

This project focused on two complementary tasks: data cleaning and visualization. I was provided with several datasets containing geographical point information from various sources, including Google Earth, the company's GPS system, Geoapify, and other geocoders. Using Python visualization tools like Plotly, I analyzed these datasets to identify duplicate geographical points and measure the distances between them. The haversine distance, calculated with the geodesic library from Geopy, was instrumental in detecting and addressing data inconsistencies. This process ensured that each geographical point was unique and accurately represented, enhancing the overall quality of the dataset.

Then, in a second part of the project, I developed a method to project package deliveries by postal code in Mexico City using demographic data from the 2020 Census. I utilized detailed population data segmented by block and geographic code, which was mapped to specific postal codes. The analysis focused on three key variables: total population, employed population, and population with internet access.

The total population was broken down into various age ranges, and separate metrics were used to determine employment levels and internet access. I performed a weighted sum of these variables by postal code, with adjustable weights to explore different scenarios. The weighted percentage for each postal code was calculated to determine the proportion of packages each code should receive. The results were visualized on a map of Mexico, using available geographic layers, with postal codes containing incomplete data represented differently to ensure clarity.
