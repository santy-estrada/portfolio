Personal Management Web Application

Web application with fully integrated front and back end

Project Details

A Jhon-Wick-Inspired web application was developed to manage personal tasks and activities. The project aimed to create a user-friendly platform. The front end was developed using Vue, while the back end was built with TypeScript. The application utilized MongoDB as the database to store user information and task data. The project also incorporated user authentication and authorization features to ensure data security and privacy.

The project involved creating a web application with a structured user management system, supporting multiple clearance levels. The application was designed to accommodate four distinct user roles: admins, managers, and two types of regular users, each with specific permissions and functionalities. Admins had the highest level of access, enabling them to create new entries, assign tasks, and mark tasks as completed. Managers had a more restricted role, limited to marking tasks as completed and creating new tasks, but without the ability to assign them. Regular users had access to view only their assigned tasks and could review the history of completed tasks.

The web application featured both registration and login pages, which connected to a MongoDB database to handle user data. This setup allowed for secure storage and retrieval of user information. To enhance security, passwords were obfuscated to prevent unauthorized access.
