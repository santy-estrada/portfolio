Neural Network Complexity Analysis

Data-driven system identification of a nonlinear pneumatic levitation system using NARX neural networks

Presented at the 9th IEEE Colombian Conference on Applications of Computational Intelligence (ColCACI 2025) as part of the Neural Networks and Learning Machines Technical Session.

Project Details

Completed: March 2026

This research investigates how neural network complexity influences nonlinear system identification. A feedforward multilayer perceptron was implemented within a NARX (Nonlinear AutoRegressive with eXogenous inputs) framework to model the dynamics of a pneumatic levitation system using only experimentally collected input-output data.

The study compares multiple neural network architectures while keeping the training conditions constant, allowing the influence of model complexity on prediction accuracy and generalization to be evaluated systematically.

Methodology & Results

A database of 4000 experimental samples obtained from a pneumatic levitation system was used to train and validate the neural models, using Matlab for the black-box nonlinear system identification. The architectures ranged from compact single-hidden-layer networks to deeper two-layer configurations, allowing a direct comparison between representational capacity and predictive performance. A total of 12 architectures were evaluated, with the number of neurons in each hidden layer varying from 1 to 10.

The experimental results demonstrated that increasing network size does not necessarily improve identification accuracy. A moderate (5,5) architecture achieved the best compromise between prediction error, generalization, and stability during recursive free-run simulations, highlighting the importance of selecting an appropriate model complexity rather than simply increasing the number of neurons.
